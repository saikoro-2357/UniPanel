# UniPanel（ゆにぱねる）

マス目を塗って、Unicodeで知らん文字を出すおもちゃ。

## ふつうのあそびかた

タテ1列の4マスを上から見て斜線=1、空白=0として16進数に変換します。出来上がった16進数4桁をユニコードにて変換したものがマス目の隣に表示されます。

RANDOMを押すと盤面の塗られ方がランダムに変わります。
COLORで背景色の設定が出来ます。
COPYで今表示されている文字をクリップボードにコピーできます。
XPOSTで現在の文字と背景色の設定でのURLを共有することができます。

## TARGET MODE

TARGETを押すことでTARGET MODEにすることができます。
このモードはお題の文字を目指してマス目を手当たり次第に切り替えて探索するモードです。
お題の文字は現在の文字の後ろにうっすらと表示されます。
HINTを使うと16マスの内ランダムな1マスの様子がわかります。
TARGET MODEをやめる時はメニューが出ている状態でもう一度TARGETのボタンを押してください。
